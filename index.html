What is PandoraBox?

Pandora is Windows 11 distro, that has been modified by my self, with the aim to boost performance without compromising the security.

It has all forensic and reversing tools that i find usefull to carry on investigations, it's important to higlight that PandoraBox IS NOT A SANDBOX but it can be configured and use as a sandbox with out any problem, but it's recommenden to use the sandbox tools that are inside.

This ISO can be mounted on VirtualBox, VmWare, Hyper-V, Ventoy...
