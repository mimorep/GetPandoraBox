# OS normal installation

Follow the next steps in order to install PandoraBox OS as your main system OS.

## 1. Download rufus

You will need to make a USB booteable, our recomendation is to directly use rufus, you can download it from it's official site:

```
https://rufus.ie/
```

## 2. Make your USB booteable

Select your desired USB on the rufus app, you will not require to activate the bypasses for Windows 11, PandoraBox manage it's by it self.

## 3. Burn PandoraBox OS on the USB

Select the PandoraBox ISO file and burn it on your USB.

## 4. Boot your machine from USB (BIOS)

The letter to boot from BIOS depends on your machine vendor, look online for the button that will open the BIOS for you, the most common are:

|Vendor| Key|
| ----------- | ----------- |
|ASUS| F2 / Delete|
|ASUS ROG| F2 / Delete|
|Acer| F2 / Delete|
|Dell| F2|
|HP| F10 / Esc|
|Lenovo| F2 / F1|
|MSI| Delete|
|Gigabyte| Delete|
|ASRock| F2 / Delete|
|Razer| F1 / Delete|
|Samsung| F2|
|Sony| F2|
|Toshiba| F2 / Esc|
|Fujitsu| F2|
|Surface| Volume Up + Power|
|Intel| F2|
|Biostar| Delete|
|EVGA| Delete|

## 5. Let PandoraBox do its magic

Just wait for the PandoraBox installers, to complete the installation for you
