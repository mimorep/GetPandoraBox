# Ventoy installation

## 1. Install Ventoy from the official site

Just download the ventoy tool from the official site:

```
https://www.ventoy.net/en/download.html
```

## 2. Download PandoraBox OS ISO

Download PandoraBox OS ISO from this website.

## 3. Install Ventoy on your USB

Follow the official steps and execute the installer for your USB media.

## 4. Drag & Drop the ISO

When the USB is ready, just drag & drop the .iso to the remobable media.

## 5. Disable secure boot

Enter the BIOS of your machine, and remove the secure boot option, othewise Ventoy will not be able to boot. Then just change the boot order to boot first from your USB.

The letter to boot from BIOS depends on your machine vendor, look online for the button that will open the BIOS for you, the most common are:

|Vendor| Key|
| ----------- | ----------- |
|ASUS| F2 / Delete|
|ASUS ROG| F2 / Delete|
|Acer| F2 / Delete|
|Dell| F2|
|HP| F10 / Esc|
|Lenovo| F2 / F1|
|MSI| Delete|
|Gigabyte| Delete|
|ASRock| F2 / Delete|
|Razer| F1 / Delete|
|Samsung| F2|
|Sony| F2|
|Toshiba| F2 / Esc|
|Fujitsu| F2|
|Surface| Volume Up + Power|
|Intel| F2|
|Biostar| Delete|
|EVGA| Delete|

A Ventoy panel will be prompted, just select the PandoraBox ISO.

## 6. Let PandoraBox do its magic

Just wait for the PandoraBox installers, to complete the installation for you
